# OSProj1
A tri-program that communicates to other programs via pipes
1. Driver.py - Holds the main cmds for user inputs with the following commands:
   - Password - sets a passkey for the encryptor
   - Encrypt - allows u to encrypt a string
   - Decrypt - allows u to decrypt a string
   - History - allows u to see old strings used
         - in option 1 2 & 3, it will list 1 to end of history in which your number chooses that pass string
           0 allows for you to put a custom string of letter
  - Quit - exits all programs
2. Logger.py - takes in actions and loggs them in to a logger file
3. Encryptor - takes in a password to set a passkey for encryption and decryption or can encrypt or decrypt a string
